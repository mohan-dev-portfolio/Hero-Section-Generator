# Hero Section Generator

A lightweight, frontend-only workbench for prototyping portfolio headers. Implemented in a single `index.html` file using raw DOM manipulation and CSS variables.

## Features
- Live editing and preview of hero/portfolio headers
- Customizable text, colors, and layout
- No frameworks or build tools required
- All logic in a single HTML file (with separate CSS and JS for maintainability)

## Usage
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Use the UI to customize your hero section.
4. Copy the generated code for your own site.

## Project Structure
- `index.html` — Main app UI and logic
- `style.css` — Styles for the app and generated hero sections
- `script.js` — JavaScript for interactivity and code generation
- `test_output.html` — (Optional) Output or test file for generated hero sections
- `README.md` — Project documentation
